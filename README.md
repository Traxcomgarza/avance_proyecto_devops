## 💻 **Ejecución local:**
Para ejecutar este proyecto localmente, te recomendamos usar **2 terminales**:

1. **Frontend:**
   - Navega a la carpeta del frontend
   - Ejecuta el comando: 
     ```bash
     ng serve
     ```

2. **Backend:**
   - Navega a la carpeta del backend
   - Ejecuta el comando: 
     ```bash
     npm run dev
     ```

¡Listo! Ahora podrás ver la aplicación funcionando localmente.
